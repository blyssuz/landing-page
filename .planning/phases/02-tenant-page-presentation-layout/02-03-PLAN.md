**use-active-section.ts:**
- Custom hook: `useActiveSection(sectionRefs: Record<string, React.RefObject<HTMLElement | null>>)`.
- Returns `activeSection: string` (the currently visible section ID).
- Uses IntersectionObserver with `rootMargin: '-80px 0px -60% 0px'` and `threshold: 0` (matches existing monolith behavior, line 435).
- When an entry isIntersecting, finds matching ref key and updates activeSection.
- Cleans up observer on unmount.
- Default initial section: 'services'.

**TabNavigation.tsx:**
- Props: `tabs: { id: string; label: string; ref: React.RefObject<HTMLElement | null>; count?: number }[], activeTab: string, onTabClick: (id: string) => void`.
- Sticky horizontal tab bar: `sticky top-0 z-30 bg-white border-b border-stone-200 shadow-warm-sm`.
- Container: `max-w-[1350px] mx-auto px-4 lg:px-6`.
- Nav: `flex gap-1 overflow-x-auto scrollbar-hide`.
- Each tab button:
  - Click handler: calls `tab.ref.current?.scrollIntoView({ behavior: 'smooth', block: 'start' })` then `onTabClick(tab.id)`.
  - Active: `text-stone-900 font-semibold`. Inactive: `text-stone-500 font-medium hover:text-stone-700`.
  - Font size: `text-[13px] lg:text-[15px]`.
  - Review count badge: if `tab.count !== undefined && tab.count > 0`, show small count pill `bg-stone-100 text-stone-500 text-xs px-1.5 py-0.5 rounded-full`.
- Active indicator: `motion.span` with `layoutId="activeTabIndicator"`, `bg-primary h-[2px] rounded-full` at the bottom of the active tab. Transition: `springs.snappy` (per user decision -- spring-animated indicator).
- Tab order: Services | Team | Reviews | About (per user decision). But tabs are passed as props, so the parent controls order and visibility.
- `'use client'` directive.

**WorkingHours.tsx:**
- Props: `workingHours: Record<string, { start: number; end: number; is_open: boolean }>, dayNames: Record<string, string>, closedLabel: string, title: string, openUntilText?: string, variant?: 'inline' | 'collapsible'`.
- Two modes via `variant` prop (per research recommendation):
  - `'inline'` (desktop sidebar): Shows full weekly schedule, no collapse. Title as h4, then day rows.
  - `'collapsible'` (mobile about): Collapsed by default showing today's hours with Clock icon and ChevronDown. Tap expands to full week. AnimatePresence for height animation using `springs.gentle`.
- Day schedule: `DAY_ORDER` iteration, each row shows day name + hours or "Закрыто".
- Today's row: `bg-primary/10 rounded-lg`, text `font-medium text-primary`.
- Closed days: `text-stone-400`.
- Use `secondsToTime` from `../_lib/utils` and `getTodayKey` from `../_lib/utils`.
- ChevronDown rotates 180 degrees when expanded (per existing pattern).
- `'use client'` directive.
