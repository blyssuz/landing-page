**Avatar.tsx:**

Props:
- `src?: string` -- image URL (optional)
- `alt?: string` -- image alt text
- `name: string` -- used for initial fallback and gradient generation
- `size`: `'sm'` (w-8 h-8 text-xs) | `'md'` (w-10 h-10 text-sm, default) | `'lg'` (w-12 h-12 text-base) | `'xl'` (w-16 h-16 text-lg)
- Extends `React.HTMLAttributes<HTMLDivElement>`

Behavior:
- When `src` is provided: render `<img>` with `object-cover rounded-full` inside the container div
- When no `src`: show gradient background with first letter of `name` (uppercase), white text, centered
- Gradient generation: deterministic from name using `charCodeAt(0) % gradients.length`. Use these gradient pairs (all warm/vibrant):
  - `from-rose-400 to-orange-300`
  - `from-violet-400 to-purple-300`
  - `from-cyan-400 to-blue-300`
  - `from-emerald-400 to-teal-300`
  - `from-amber-400 to-yellow-300`
- Container: `rounded-full overflow-hidden flex-shrink-0 flex items-center justify-center`
- Add `ring-2 ring-white` for a subtle border effect when used on colored backgrounds

**Skeleton.tsx:**

Follow the research code example closely. Props:
- `variant`: `'text'` (rounded-md, default) | `'circular'` (rounded-full) | `'rectangular'` (rounded-2xl)
- `width?: string | number`
- `height?: string | number`
- Extends `React.HTMLAttributes<HTMLDivElement>`

Classes: `bg-gradient-to-r from-stone-200 via-stone-100 to-stone-200 bg-[length:200%_100%] animate-[shimmer_1.5s_ease-in-out_infinite]`
Add `aria-hidden="true"` for accessibility.
Text variant default height: `h-4`. Circular gets equal width/height if only one dimension specified.

**Input.tsx:**

Props:
- `variant`: `'default'` | `'filled'` (stone-50 background)
- Extends `React.InputHTMLAttributes<HTMLInputElement>`

Base classes: `w-full rounded-xl border border-stone-200 bg-white px-4 py-2.5 text-base text-stone-900 placeholder:text-stone-400 transition-colors duration-200`
Focus: `focus:border-primary focus:ring-1 focus:ring-primary/20 focus:outline-none`
Filled variant: `bg-stone-50 border-transparent focus:bg-white focus:border-primary`
Disabled: `disabled:opacity-50 disabled:cursor-not-allowed`

Mobile-first: minimum 16px text size on mobile (text-base = 16px).

All components: `'use client'`, forwardRef, displayName, named export, cn() for class merging.
