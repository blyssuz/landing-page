**State management:**
- `open: boolean` — chat window visibility
- `chatLocale: ChatLocale | null` — selected language (null = not selected yet)
- `messages: ChatMessage[]` — message history within this session
- `typing: boolean` — simulated typing indicator
- `flowState: FlowState` — current position in the flow

**Response generators — functions that build response text from real business data:**

1. `generatePricesResponse(services, locale, t)` — Format all services with name, duration, and price. Group by category if categories exist. Use `getText()` for service names, `formatPrice()` for prices, `formatDuration()` for duration. Example output:
   ```
   Sartaroshlik — 30 daq — 50 000 so'm
   Soqol olish — 20 daq — 30 000 so'm
   ```
   If no services, return `t.noServices`.

2. `generateServicesResponse(services, locale, t)` — Same data as prices but formatted as a simple list of service names with brief descriptions (use `getText()` on `service.description` if available). If no services, return `t.noServices`.

3. `generateLocationResponse(business, t)` — Return `business.location.address` or `t.noAddress` if not set.

4. `generateWorkingHoursResponse(business, locale, t)` — Format working hours using `DAY_NAMES[locale]` and `DAY_ORDER`. For each day, show "Dushanba: 09:00 - 21:00" or "Dushanba: Yopiq". Use `secondsToTime()` to convert start/end seconds. If no working_hours, return `t.noWorkingHours`.

5. `generateContactResponse(business, t)` — Show phone number and Instagram link if available. Format: "Telefon: +998 XX XXX XX XX\nInstagram: @username". If no contact info, return `t.noContact`.

**Flow logic:**

- On chat open: if `chatLocale` is null, show language selection (two buttons: "O'zbekcha" / "Русский")

- Main menu buttons: Prices, Services, Location, Working Hours, Contact (using `CHAT_TEXT[chatLocale]` labels)
- On menu button click: add button label as user message, show typing for random 500-1000ms, then add predefined response + "Back to menu" button
- On "Back to menu" click: add "Back to menu" as user message, show typing for 400ms, re-show main menu with buttons
