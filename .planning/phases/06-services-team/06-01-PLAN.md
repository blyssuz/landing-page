Create a new `ServicesSection.tsx` component using plain function export (not forwardRef, per v2.0 convention from P5-01).

**Props interface (ServicesSectionProps):**
- `services: Service[]` -- all services
- `locale: Locale` -- current locale
- `onBook: (service: Service) => void` -- booking callback
- `bookingServiceId: string | null` -- currently booking service (for spinner)
- `translations: { services: string; noServices: string; allPhotos: string; general: string; book: string; minute: string; hour: string; sum: string }` -- translation subset

**Internal state:**
- `activeCategory: string | null` -- null means "All"
- `expandedId: string | null` -- single expanded service ID (accordion)

**Service grouping logic (extract from TenantPage lines 51-57):**
```typescript
const groupedServices = services.reduce<Record<string, Service[]>>((acc, svc) => {
  const cat = svc.category || 'general';
  (acc[cat] ??= []).push(svc);
  return acc;
}, {});
const categories = Object.keys(groupedServices);
const displayedServices = activeCategory ? { [activeCategory]: groupedServices[activeCategory] || [] } : groupedServices;
```

**Category pills (extract from TenantPage lines 86-93):**
- Only render when `categories.length > 1` (per CONTEXT.md: hide pills when single category)
- "All" pill uses `t.allPhotos` translation key (Hammasi/Vse)
- Active pill: `bg-stone-900 text-white`; inactive: `bg-white text-stone-600 border border-stone-200 hover:bg-stone-50`
- Horizontal scroll: `flex gap-2 overflow-x-auto scrollbar-hide pb-4 -mx-1 px-1`
- No animation on category switch (instant swap per CONTEXT.md)
- Selecting a category resets `expandedId` to null (close any open row)

**Service rows -- dual mode:**

Determine expandability per row:
```typescript
const description = getText(service.description, locale).trim();
const isExpandable = description !== '';
```

**Non-expandable row (no description):**
- Row is NOT tappable (no cursor-pointer, no onClick)
- Left side: name (line-clamp-1), duration, price
- Right side: Book button always visible
- No chevron icon

**Expandable row (has description):**
- Entire row is tappable (`cursor-pointer`, onClick toggles `expandedId`)
- Left side: name (line-clamp-1), duration, price
- Right side: ChevronDown from lucide-react, rotates 180 degrees when expanded (motion.div with `animate={{ rotate: isExpanded ? 180 : 0 }}`, `transition={{ duration: 0.2 }}`)
- Book button hidden in collapsed state
- Accordion: `setExpandedId(id === expandedId ? null : id)`

**Expanded content (AnimatePresence pattern from WorkingHours.tsx):**
```tsx
<AnimatePresence>
  {isExpandable && isExpanded && (
    <motion.div
      initial={{ height: 0, opacity: 0 }}
      animate={{ height: 'auto', opacity: 1 }}
      exit={{ height: 0, opacity: 0 }}
      transition={springs.gentle}
      className="overflow-hidden"
    >
      <div className="pb-4">
        <p className="text-sm text-stone-500 mb-3">{description}</p>
        {/* Book button */}
      </div>
    </motion.div>
  )}
</AnimatePresence>
```

**Book button (both modes):**
- Outline pill style: `px-5 py-2 rounded-full text-sm font-medium border border-stone-200 text-stone-900 hover:bg-stone-50 transition-all duration-200`
- onClick: `e.stopPropagation()` (prevent row toggle) then `onBook(service)`
- Disabled when `bookingServiceId === service.id` with `disabled:opacity-70`
- Show spinner SVG when disabled (same spinner from current TenantPage code)

**Row dividers:** Use `border-t border-stone-100` on rows after the first (idx > 0)

**Empty state:** When no services at all, show centered `<p className="text-stone-400 text-sm">{t.noServices}</p>` in a `py-16 text-center` container.

**Category subheadings:** When showing all categories and `categories.length > 1`, show `<h3>` subheading per category group (capitalize, use `t.general` for 'general' category).

**Section heading:** `<h2 className="text-xl font-semibold text-stone-900 mb-4">{t.services}</h2>` at the top.

**Imports:**
- `{ useState }` from react
