**Create TeamStrip.tsx** as a plain function export (not forwardRef):

**Props interface (TeamStripProps):**
- `employees: Employee[]` -- all employees
- `locale: Locale` -- current locale (reserved for future use, e.g., localized position names)
- `translations: { specialists: string }` -- section heading

**Visibility guard:** Return `null` when `employees.length <= 1` (per CONTEXT.md: hide team section for solo employee, using `<=` not `<`).

**Section heading:** `<h2 className="text-xl font-semibold text-stone-900 mb-4">{t.specialists}</h2>`

**Avatar strip (follow PhotoStrip scroll pattern):**
```tsx
<div className="flex gap-4 overflow-x-auto scrollbar-hide -mx-1 px-1">
  {employees.map((employee) => {
    const name = [employee.first_name, employee.last_name].filter(Boolean).join(' ');
    const displayName = name || employee.position;
    const avatarName = employee.first_name || employee.position || '?';

    return (
      <div key={employee.id} className="flex flex-col items-center flex-shrink-0">
        <Avatar
          name={avatarName}
          size="xl"
          className="w-14 h-14 text-sm"
        />
        <p className="text-xs font-medium text-stone-900 mt-1.5 text-center line-clamp-1 w-16">
          {displayName}
        </p>
        {name && employee.position && (
          <p className="text-[11px] text-stone-500 text-center line-clamp-1 w-16 mt-0.5">
            {employee.position}
          </p>
        )}
      </div>
    );
  })}
</div>
```

**Key details:**
- Avatar at 56px (w-14 h-14) with text-sm for initial letter sizing
- Name text: text-xs font-medium, max width w-16 (64px), line-clamp-1
- Role text: text-[11px], shown only when both name AND position exist
- NOT tappable (no onClick, no cursor-pointer per CONTEXT.md)
- No 'use client' needed unless Avatar requires it (Avatar uses className only, no client hooks needed in TeamStrip itself -- but add 'use client' since Avatar is a client component)

**Imports:**
- `Avatar` from '@/app/components/ui/Avatar'
- `Employee` type from '../_lib/types'
- `Locale` type from '../_lib/types'
